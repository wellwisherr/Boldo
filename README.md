
# Bootstrap 4.4.1 Boilerplate - Basic

This is a Bootstrap 4.4.1 Boilerplate with Gulp 4+. Sass, browser-sync.

Install $ `npm install`
Init $  `npm init`
Start $ `gulp serve`
Prod $ `npm run prod`
